## Разблокировка клиента
[POST /clients/{client_id}/unblock](https://github.com/yn96k/client-block-service/blob/main/client-block-service-v1.openapi.yaml)

### Входные параметры 
| Название |Тип параметра| Описание |
|:--|:---|:---|
|clientId|path|Идентификатор клиента|

### Алгоритм метода:

1. Обновляет поля block_type, blocked_at, comment в таблице client_block БД client:
   
```sql
UPDATE client_block
SET block_type = "unblocked"
SET blocked_at = NULL
SET comment = NULL
WHERE client_id = {client_id из входных параметров}
```
  1.1. Если запись отсутствует в БД - вернуть ошибку с кодом 404 <br>
2. Завершить работу алгоритма, вернув успешный ответ с кодом 204.
